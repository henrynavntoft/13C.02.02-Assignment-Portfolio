Responsive Design

In this assignment, I applied CSS Flex and Grid techniques to create a responsive layout for a mockup landing page. I started by examining the HTML in the browser and identifying components and layouts within the code. I used Flex and Grid to arrange the elements, referring to the HTML and CSS used throughout the course for guidance.

I added images to all spaces required using resources from stock image websites like Adobe Stock, Shutterstock, and Pexels. I adjusted the size of the images with width, height, min-width, and min-height, using rem, vh, and vw for responsive units. I also set background-repeat, background-size, and background-position as needed.

I made sure the design looked great on both desktop and mobile devices. As an extra step, I added a font hierarchy using rem units for responsiveness and applied a color scheme to enhance the design's visual appeal.
