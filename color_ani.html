Colors & Animations

In this assignment, I enhanced a mockup landing page by applying a color palette, creating a dark theme, and adding CSS animations. I began by creating a color palette using Huemint, validating the contrast with Adobe's Contrast Analyzer, and applying it to the website.

Next, I developed a dark theme following the tutorial and incorporated extra tips from the material. Lastly, I added CSS animations to create a more engaging design.

The final result is a visually pleasing and engaging landing page showcasing my understanding of color palettes, dark themes, and CSS animations.
